This is not the end

As you can see in this part of my code, my main class represents the beginning of something bigger. It’s not just a component with its measurements; in the process, I created the Builder system. I felt the need to implement something in production, not just at the end of the process. What does the Builder do? It creates the machining process that can be used on CNC machines for all structural and visual details. In this case, I created a simple base for a cabinet and designed the basic process for a minifix, which helps to attach the base to the side piece. I'm referring to the “minifix” and also included the “buiSerraCopo,” a test for larger holes that we typically make for electrical wires, included here just as an example. In this case, I used the main class, but I intend to use it in an implementation with the system screen. This part of the system can be used in engineering to create budgets with cabinet information, and it allows for the production of cabinets of any size with the engineering fully completed.

Json to Real Life

After developing the basic Builder system, I created the capability to perform a square CNC operation. In this class, you can input the initial positions on the X and Y axes, as well as the width and height. This operation is commonly used for built-in ovens and stoves, but you can use your imagination to create any type of work. In this implementation, I searched for some files that I knew had implementations with CNC machines. I'm not certain if they are truly different, but the API I observed uses a simple JSON format. What is the real challenge for this system to become more extensive and functional? I don't have access to any CNC machines to connect with the API. I know that all the necessary files need to specify the face on which the CNC code will be executed, as some parts have holes for structural engineering on both sides. However, how do I determine which side it is?

The Json

As you can see, I created the capability for the MDF class to receive a list of Builder objects. This handles all the engineering work, allowing me to extract the JSON and use it with any CNC machine (as I believe most CNC machines operate on this principle). All these processes are performed using a milling cutter, which we use for most of our tasks. However, we also have another tool in the CNC machine, the saw, which plays a significant role in the work. Most of the base and side pieces require operations with the saw, and I am uncertain about how to pass the properties needed to use it.
